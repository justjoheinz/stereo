stereo
by Charlie Roberts & Angus Forbes

Stereographic library for Processing (based on code by Paul Bourke)

Dependencies

- core.jar from the most recent version of processing. - jogl.jar, gluegen-rt.jar (both from the stable version of jogl1.1)

Install

Netbeans 6.8 project. Link to the three jars in the lib directory and build. Make a directory under your processing sketches directory like so: myprocessingsketches/libraries/stereo/library and place dist/stereo.jar into it. Run the demo .pde file and modify as needed.
